# Taco Clicker - Desktop Version

Desktop version of the Taco Clicker game, built with Electron.

## About

This repository contains only the essential configuration files for building the desktop version. Game files are added during the release process for security reasons.

## Releases

Check the [Releases](https://github.com/tacovr-12/taco-clicker/releases) page to download the latest version of the game.

## Auto Updates

The game will automatically check for and install updates when launched.
